level 3.1 - Adding & Removing Classes

-- addClass() method add class/es to the selected elements.
-- removeClass() method remove class/es to the selected elements.
-- toggleClass() method -  between addClass() -and- removeClass() classes.
    remove a Class - when class exist in an element.
    add a class - when class not exist in an element.
--- example: <"example-3.1.1.html"> - addClass( ) method - addClass/removeClass/toggleClass class from <div> element  (class name "header")
